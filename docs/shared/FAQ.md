# Frequently asked questions

## Does WAJetsamFix change Jetsam globally?

No. The supported policy is intentionally limited to the exact WhatsApp notification `ServiceExtension` and an exact supported 24 MB candidate.

## Are the two editions interchangeable?

No. RootHide Edition and Dopamine Edition are separate packages for different jailbreak environments. Do not install the RootHide package on conventional Dopamine.

## Why is only the newest stable package downloadable?

Older Releases are kept as changelog history, not as recommended installation sources. This reduces accidental installation of obsolete builds while preserving the project's evolution.

## Does a successful installation prove compatibility?

No. Installation, build range and field validation are different claims. Consult the compatibility document for the edition being used.

## Will every WhatsApp update be supported automatically?

No. Changes to the extension path, identity, structure or original memory policy require review. Unknown conditions pass through unchanged.
