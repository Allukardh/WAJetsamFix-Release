# Security and package integrity

Download packages only from the official WAJetsamFix public Releases page.

Before any RootHide conversion step, verify the SHA-256 checksum of the original downloaded package against `CHECKSUMS.txt` and the checksum published in the Release notes.

The RootHide Patcher output may have a different checksum because it is a converted package. The official checksum therefore applies to the original Release asset before conversion.

The public repository intentionally contains documentation and release packages only. Source code, internal test records, private architecture notes and development history are not part of public distribution.
