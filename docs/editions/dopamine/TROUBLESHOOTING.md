# Dopamine Edition troubleshooting

No public Dopamine Edition package currently exists, so no public troubleshooting command should be applied to a private or unofficial build.

When an approved RC or stable package is available, troubleshooting will be edition-specific and will require:

- exact package version;
- device and iOS version;
- Dopamine and ElleKit versions;
- WhatsApp version;
- required Choicy plugin state;
- selected memory profile and live-loaded profile;
- observed ServiceExtension peak when the feature is available;
- complete output from the documented Dopamine diagnostic command.

RootHide Edition status output and recovery steps must not be used to judge conventional Dopamine state.

Future diagnostics may display locally detected relevant tweak versions as informational context. That display will not imply endorsement or universal compatibility.

Private development reports and raw internal evidence must not be posted publicly without review.
