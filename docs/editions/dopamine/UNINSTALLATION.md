# Dopamine Edition uninstallation

No public Dopamine Edition package is currently available.

Removal commands must be published only with an exact approved package identity so users do not accidentally remove or damage the independent RootHide Edition.

A future approved Dopamine RC or stable package will include:

- normal package-manager removal;
- terminal removal using the exact package ID `com.allukardh.wajetsamfix.dopamine`;
- Userspace Reboot guidance;
- Safe Mode recovery;
- preference and helper cleanup;
- verification that the Dopamine dylib and tools were removed;
- clean-reinstall instructions when appropriate.

Private Alpha bundles retain their own exact recovery procedure and are not public installation sources.
