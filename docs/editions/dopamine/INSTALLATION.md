# Dopamine Edition installation

No public Dopamine Edition package, RC or installation procedure is available.

Do not install the RootHide Edition package on conventional Dopamine. The two editions use different package identities and jailbreak integration paths.

When a Dopamine package is explicitly approved, this page will identify:

- exact package filename and package ID;
- version and SHA-256;
- validated Dopamine, ElleKit, iOS and WhatsApp scope;
- required Choicy exclusions;
- selected or automatic memory-profile behavior;
- Userspace Reboot procedure;
- verification command and expected result;
- Safe Mode and recovery steps;
- complete uninstallation procedure.

Private Alpha existence or a roadmap entry does not authorize installation or redistribution.
