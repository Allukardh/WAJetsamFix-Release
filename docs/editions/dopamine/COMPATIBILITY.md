# Dopamine Edition compatibility

## Intended environment

- conventional rootless Dopamine 2;
- regular ElleKit tweak injection;
- arm64 and arm64e targets where supported by Dopamine;
- WhatsApp notification `ServiceExtension` as the eventual exact target.

## Current validation state

No device combination has been approved yet. A volunteer tester using conventional Dopamine will validate the diagnostic Alphas through objective terminal commands and complete reports.

No compatibility promise will be published until real-device evidence exists.
