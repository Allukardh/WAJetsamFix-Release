# RootHide Edition installation

1. Download the current stable `.deb` from the official public Release.
2. Verify the checksum of the original downloaded package against `CHECKSUMS.txt` and the Release notes.
3. Process the original package with **RootHide Patcher**. This conversion is mandatory.
4. Install the converted package through Sileo.
5. Open the Dopamine 2 RootHide app and manually perform a **Userspace Restart**.

Do not install the original, unconverted package directly on RootHide.

Sileo does not currently request the required Userspace Restart automatically. Perform it manually after installation.
